🎬 Movie Success Prediction & Sentiment Analysis


 📌 Project Overview

This project predicts movie success based on features like budget, popularity, and ratings. It also performs sentiment analysis using movie descriptions.


🛠️ Tools Used

- Python

- Pandas, NumPy

- Scikit-learn

- NLTK (VADER)

- Matplotlib, Seaborn


📊 Features

- Data Cleaning & Preprocessing

- Sentiment Analysis

- Machine Learning Model (Logistic Regression)

- Data Visualization


📈 Results

- Achieved ~75–85% accuracy

- Positive sentiment correlates with higher success


 📁 Dataset

TMDB 5000 Movies Dataset


 🚀 How to Run

1. Open notebook.ipynb in Jupyter/Colab

2. Run all cells
